# Fashion Product Recommendation using CLIP

## Overview
This project implements a fashion product recommendation system using OpenAI's CLIP (Contrastive Language-Image Pretraining) model. It generates image embeddings, identifies duplicate products, clusters similar items, and performs reverse product search using text queries.

## Features
- Image embedding generation using CLIP
- Duplicate product detection
- Product clustering based on visual similarity
- Reverse product search (Text → Image)
- Unique product catalog generation
- Visualization of similar products

## Technologies Used
- Python
- PyTorch
- OpenAI CLIP
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- PIL (Pillow)
- Kaggle Notebook

## Dataset
Fashion product image dataset available on Kaggle.

## Project Workflow
1. Load dataset
2. Preprocess images
3. Generate CLIP embeddings
4. Cluster similar products
5. Detect duplicate images
6. Build unique product catalog
7. Perform text-based product search

## Results
- Successfully generated image embeddings using CLIP.
- Identified visually similar and duplicate products.
- Created a unique product catalog.
- Enabled semantic text-to-image search.

## How to Run
1. Open the notebook in Kaggle or Jupyter Notebook.
2. Install the required libraries.
3. Download the dataset.
4. Update the dataset path if necessary.
5. Run all notebook cells sequentially.
